# HR TYPING ASSESSMENT & RECRUITMENT MODULE

Build a complete HR Assessment System for the Traffic Peak platform.

This module should allow recruiters, companies, training institutes, and schools to evaluate typing skills through shareable assessment links.

## OVERVIEW

The system should allow:

* HR users to create typing assessments
* Generate unique assessment links
* Share links with candidates
* Track candidate performance
* Generate reports
* Rank candidates automatically
* Download results in PDF and Excel formats

---

## HR DASHBOARD

Create a dedicated HR dashboard.

Features:

### Overview Cards

* Total Assessments
* Total Candidates
* Average WPM
* Average Accuracy
* Pass Rate
* Active Assessments

---

## CREATE ASSESSMENT

HR can create a typing assessment.

Fields:

* Assessment Name
* Company Name
* Job Position
* Assessment Description
* Duration
* Difficulty Level
* Language
* Passing WPM
* Minimum Accuracy
* Number of Attempts Allowed
* Expiry Date

Example:

Company: ABC Technologies

Position: Data Entry Operator

Duration: 5 Minutes

Passing WPM: 40

Minimum Accuracy: 95%
